# 👨‍🍳 Chef Claude

A beginner-friendly React app that generates recipes from available ingredients using **Claude AI (via Puter API)**.

---

## 🚀 Features

- Add ingredients dynamically
- Generate AI-based recipes
- Clean UI with ingredients + instructions
- Built using React + Vite

---

## 🛠️ Tech Stack

- React (Hooks)
- Vite
- Puter API (Claude AI)
- CSS

---

## ⚙️ Setup

```bash
git clone https://github.com/mohitt10/chef-claude.git
cd chef-claude
npm install
npm run dev

```
